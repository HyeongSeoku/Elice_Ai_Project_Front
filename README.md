## 2. 작업 내용

기획 단계: 구체적인 설계와 지표에 따른 프로젝트 제안서 작성, 피그마를 이용한 와이어 프레임 작업

개발 단계: 효율적인 작업을 위한 프론트엔드 전반적인 사용자의 편의를 위한 화면 구성 , 백엔드와의 효과적인 데이터 교환 

수정 단계: 코드의 가독성 및 효과적인 로직을 위해 리팩토링 , 프론트엔드 배포
